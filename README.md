## 特点 :anchor:

- 在线版个人简历预览,支持导出为 pdf
- 3d graphic 信息预览.支持截图下载
- 基于 react 构建

## 使用 :key:

- 下载

git clone 或 下载源码包

```bash
$ git clone 仓库地址
```

- 安装依赖

```bash
$ npm install or npm i
```

- 运行

```bash
$ npm run carco
```
